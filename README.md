# 🎨 Pick-me

### 개발자 구인구직 사이트 (백 엔드 프로젝트)

<br/>

## 상세 소개  
&nbsp;코딩온 교육기관을 통해 진행한 첫번째 프론트엔드 프로젝트입니다.  
&nbsp; 어떤 영상에서 무명 모델 또는 배우들이 회사들을 돌며 회사 문 앞에 놓여진 바구니에 포트폴리오를 두고 가면 회사에서 필요한 역할에 맞는 사람들에게 연락을 하고 면접을 보는 장면이 있었다.  
&nbsp; 여기서 아이디어를 착안하여 개발자 역시 본인 기술 및 경력 등의 정보를 적어 올려두면 회사에서는 우리가 필요한 기술을 카테고리에서 선택하여 분류한 후 선택을 하는 일종의 구인구직 웹 사이트를 제작하였다.
